# 🌤️ Weather App

A full-stack weather application that allows users to search for real-time weather data by city. This project demonstrates a simple client-server architecture and API integration.

##  Features
- Search weather by city name  
- Fetch real-time data from an external API
- Add your full wardrobe to know what to pack to each city
- Check air polution
- Browse Popular locations
- Dynamic UI updates  
- Clear separation between frontend and backend  

##  Tech Stack
**Frontend:** HTML, CSS, JavaScript  
**Backend:** Node.js, Express  
**Other:** Axios, External Weather API

##  Getting Started

1. Clone the repository  
git clone https://github.com/seanl141/weather-app.git  

2. Navigate to the server  
cd weather-app/Server  

3. Install dependencies  
npm install  

4. Run the server  
node server.js  

5. Open the client  
Open Client/index.html in your browser
